# Chinaclaw MiniMax Integration







Configure and run ChinaClaw full research pipeline using MiniMax API.







## Overview







This skill provides the correct configuration for using MiniMax's API



with the Chinaclaw research workflow, avoiding common configuration errors.







## Required Environment Variables







- `MINIMAX_API_KEY`: Your MiniMax API key



- `MINIMAX_BASE_URL`: API base URL (default: https://api.minimax.chat)







## Usage







Use this skill when setting up Chinaclaw with MiniMax backend.
